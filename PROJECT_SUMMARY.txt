================================================================================
                          REPOLICENSE-CLI PROJECT SUMMARY
================================================================================

PROJECT: repolicense-cli
DESCRIPTION: Command-line tool to help choose open-source licenses
LANGUAGE: Zig
VERSION: 1.0.0
MINIMUM ZIG VERSION: 0.13.0

================================================================================
IMPLEMENTATION STATUS: ✅ COMPLETE
================================================================================

The entire logic from the repolicense web application (github.com/lucasfth/repolicense)
has been successfully transferred to this Zig CLI application.

KEY FEATURES:
- Interactive question/answer interface
- Complete decision tree with 17+ licenses
- Navigation commands (back, reset, quit)
- Elaborations for each question
- GitHub API links for license information
- No external dependencies
- Cross-platform support (Linux, macOS, Windows)

================================================================================
BUILD & RUN
================================================================================

Prerequisites:
  - Zig 0.13.0 or later

Build:
  $ zig build

Run:
  $ ./zig-out/bin/repolicense

Or build and run:
  $ zig build run
